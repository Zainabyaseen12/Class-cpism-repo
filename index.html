HTML

DEFINITION

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web. HTML consists of a series of elements.HTML elements tell the browser how to display the content.

USAGE OF HTML

HTML is used to create webpages but does experience limitations when it comes to fully responsive components. Therefore, HTML should only be used to add text elements and structure them within a page. For more complex features, HTML can be combined with cascading style sheets (CSS) and JavaScript (JS).

ATTRIBUTES AND ENTITIES

In HTML, attributes provide additional information about elements and are specified within the opening tag as name/value pairs, like href="https://example.com" in an anchor tag. Common attributes include class, id, and style. On the other hand, entities are special codes used to represent characters that have a specific meaning in HTML or are not easily typed, such as   for a non-breaking space and < for the less-than sign. Together, attributes and entities enhance the functionality and readability of HTML documents.

TYPES OF ATTRIBUTES

HTML attributes can be categorized into several types based on their function. Global attributes like id, class, and style apply to any element, providing unique identifiers or styling. Event attributes such as onclick and onmouseover trigger JavaScript functions in response to user actions. Form attributes like action and method define form submission behavior, while media attributes (e.g., src and alt) enhance the functionality of images and audio/video elements. Lastly, link attributes in anchor tags, such as href and target, specify the URL and how it opens. Together, these attributes enhance the interactivity and functionality of web pages.
